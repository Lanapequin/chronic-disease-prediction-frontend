# Chronic Disease AI - Frontend Setup

Este documento describe el proceso paso a paso para configurar y ejecutar el frontend en Next.js.

## Requisitos Previos

- Node.js 16.x o superior instalado
- npm (gestor de paquetes de Node.js)
- Git (opcional, si se clona el repositorio)

## Instalación y Configuración

### 1. Instalar las dependencias

Instala todas las dependencias necesarias del proyecto:

```bash
npm install
```

Este comando leerá el archivo `package.json` e instalará todas las bibliotecas requeridas.

### 2. Ejecutar el servidor de desarrollo

Inicia el servidor de desarrollo de Next.js:

```bash
npm run dev
```

El servidor estará disponible en: `http://localhost:3000/`
